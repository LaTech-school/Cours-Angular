1) Installation de node

    Télécharger NodeJs : https://nodejs.org/en/

2) Test de NodeJS 

    En ligne de commande
    $ node -v
    > v10.16.3

    $ npm -v
    > 6.14.6

3) Installation de Angular (ng)

    $ npm install -g @angular/cli
    > share stats (y/N)


4) Création du projet

    $ ng new <project-name>
    > Would you like to add Angular routing? (y/N)
    ? Which stylesheet format would you like to use? 
    ❯ CSS 
    

5) Cibler le répertoire du projet 

    $ cd <project-name>


6) Démarrage du serveur de développement

    $ ng serve

    Ouvrir le navigateur : http://localhost:4200

    Arret du serveur : ctrl + c



7) Creation d'un Module

    $ ng generate module book
    $ ng g m book
